## 3. ADR Template

```
ADR-NNN: <Title>
Status: Proposed | Accepted | Superseded | Deprecated
Date: YYYY-MM-DD
Context:
  <What problem are we solving? What's the current state?>
Options Considered:
  - Option A — pros / cons
  - Option B — pros / cons
Decision:
  <Selected option and rationale>
Consequences:
  <Positive, negative, follow-on actions>
Linked Artifacts:
  <D-IDs that depend on this decision>
```
